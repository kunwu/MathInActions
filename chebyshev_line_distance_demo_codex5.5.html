点到直线的识别距离最小值

在平面直角坐标系中，定义两点的“识别距离”为横坐标差与纵坐标差中的较大者。 已知 D(0, 1)，直线上的动点为 P(m, 3m/4 + 3)。本演示求 max(|m|, |3m/4 + 2|) 的最小值，并展示状态如何从“纵向差主导”切换到“横向差主导”。

直线：y = 3x/4 + 3；固定点：D(0, 1) 纵向差决定识别距离
横向距离 |Δx| 0 D.x 与 P.x 的差
纵向距离 |Δy| 0 D.y 与 P.y 的差
识别距离 max(|Δx|, |Δy|) 0 取两者中较大的数值
最小值与取值点 8/7 当 m = -8/7 时，P = (-8/7, 15/7)，此时 |Δx| = |Δy| = 8/7。
-4.200

推导过程

1 把直线上的任意点写成 P(m, 3m/4 + 3)。
2 相对 D(0, 1)，横向差为 |m|，纵向差为 |3m/4 + 2|。
3 识别距离是两者最大值；想让最大值尽量小，临界时两者相等。
4 在最优区间内，-m = 3m/4 + 2，得到 m = -8/7，最小识别距离为 8/7。
