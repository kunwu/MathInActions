直角三角形动态示意图

点 M 以速度 1 从 A 出发沿 A-C-B 运动，点 N 以速度 3 从 B 出发沿 B-C-A 运动。时间可手动拖动，也可自动播放。

题目描述

已知一条水平直线 l，在其上方有直角三角形 ABC。其中 C 为直角顶点，且点 C 在直线 l 上；斜边 AB 位于直线 l 上方。已知 AC = 7，BC = 11。

设边 AC 与水平方向的夹角为 α，初始值为 35°。

点 M 从点 A 出发，沿折线 A → C → B 运动，速度为 1；点 N 从点 B 出发，沿折线 B → C → A 运动，速度为 3。点 M 到达 B 时停止，点 N 到达 A 时停止。

过点 M 向直线 l 作垂线，垂足为 E；过点 N 向直线 l 作垂线，垂足为 F。

研究运动过程中 ME、NF、MC、NC 的变化情况，以及 MC 与 NC 何时相等。

N 位置 B 点
